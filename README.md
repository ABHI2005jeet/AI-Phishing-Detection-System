# AI-Based Phishing Detection System

A hybrid phishing detection system using Machine Learning, Flask backend, Chrome Extension, and Offline Dashboard.

---

## 📌 How to Clone This Repository

### Step 1: Copy Repository URL
Click the green **Code** button and copy the HTTPS URL.

### Step 2: Clone in Your System

Open terminal and run:

git clone https://github.com/ABHI2005jeet/AI-Phishing-Detection-System.git

###Step 3: Move into Project Folder

cd AI-Phishing-Detection-System




 [*****]
( How to Fork This Repository)->

Click the Fork button (top right of this page)

GitHub will create a copy in your account
Clone your forked repository:

git clone https://github.com/YOUR-USERNAME/AI-Phishing-Detection-System.git





⚙️ How to Run in Your System
1️⃣ Create Virtual Environment
python -m venv venv
2️⃣ Activate Virtual Environment

Windows:

venv\Scripts\activate

Mac/Linux:

source venv/bin/activate
3️⃣ Install Dependencies
pip install -r requirements.txt
4️⃣ Run Application
python app.py

Open in browser:

http://127.0.0.1:5000
